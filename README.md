# Weather_Vis_2025 全球气象大数据监测平台

🚀 基于 Python Flask + ECharts + MySQL 构建的高级气象可视化系统。

## ✨ 项目亮点
- **全景玻璃态 UI**：极致的 Apple Style 视觉设计，支持四季动态背景切换。
- **动态热力地图**：全国 3000+ 城市实时气温热力分布，支持拼音模糊搜索。
- **多维数据分析**：点击城市可查看 2025 全年平均气温走势及极端高温/低温明细。
- **完善的权限系统**：包含普通用户改密申请及管理员专属用户管理后台。

## 🛠️ 技术栈
- **后端**：Python 3.12 / Flask / PyMySQL
- **前端**：JavaScript / ECharts 5.x / Font-Awesome
- **数据库**：MySQL 8.0+

## 📦 快速启动
1. 安装依赖：`pip install -r requirements.txt`
2. 配置数据库：修改 `app.py` 中的 `DB_CONFIG` 映射至你的数据库。
3. 启动服务：`python app.py`
4. 访问地址：`http://127.0.0.1:5000`
